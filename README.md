# TypeScript Studies

This is an open repository containing studies done throughout the course from Udemy [Mastering TypeScript - 2025 Edition](https://www.udemy.com/course/learn-typescript/), created by [Colt Steele](https://github.com/Colt).

The course was taken from Octorber 31st, 2025 until this day (November 7th).

## What I've Learned

- General Type Annotations
- Type inference and the Any type
- Functions with parameter annotations and return type annotations
- Void and Never Types
- Object types, type aliases, nested objects, properties (optional, readonly) and intersections
- Array types and multidimensional arrays
- Union types, type narrowing, union types with arrays and literal types
- Tuples and enums
- Differences between enums and created objects
- Interfaces, properties (readonly and optional), methods and methors parameters
- Reopening, extending and multiple inheritance Interfaces
- Differences betweehn type aliases vs interfaces
